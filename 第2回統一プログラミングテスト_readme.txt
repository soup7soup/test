■第1問　Google Sheets APIを利用してデータを抽出する
https://github.com/soup7soup/test/tree/master/no1s/google_sheets_api

実行する時、該当フォルダに移動して、
python [該当ファイル名] [API_KEY]
この形で入力して実行してください。

■第2問　GUIを用いたアニメーションとイベント駆動処理の作成
https://github.com/soup7soup/test/tree/master/no1s/MarioJS

GITをCLONEして、eclipseでMarioJSフォルダを指定してインポートします。
そしてから、index.htmlを右クリックして、Open With -> Web Browserで実行します。
